# Online Payment Fraud Detection using Machine Learning

This project uses Machine Learning techniques to identify fraudulent online payment transactions. Multiple machine learning models were trained and evaluated to determine the most effective approach for fraud detection.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- One-Hot Encoding for categorical variables
- Fraud detection using Machine Learning
- Model comparison and evaluation
- ROC-AUC score analysis
- Confusion Matrix visualization

## Technologies
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-Learn, XGBoost

## Models Used
- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier

## Evaluation Metrics
- ROC-AUC Score
- Confusion Matrix

## Results
XGBoost achieved the best performance for detecting fraudulent transactions.
